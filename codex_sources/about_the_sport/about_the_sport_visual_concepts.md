## 3. Difference Between Skydiving and BASE Jumping

### Side by Side Comparison

| Skydiving | BASE Jumping |
|------------|------------|
| Aircraft | Fixed Object |
| ~4000m Exit Altitude | ~50–1000m Exit Altitude |
| Reserve Parachute | Usually No Reserve |
| Large Landing Areas | Often Small Landing Areas |
| More Time for Decisions | Very Limited Time |

**Purpose**

* High educational value
* Easy to understand
* Mobile friendly layout
